A self-contained Instagram browser tool providing anonymous access to the site (when possible).

Since no user authentication is taking place, some features will be limited or missing compared to Instagram's own interface. For instance, only logged-in users on the Instagram domain and certified Instagram apps are granted access to the following data
- list of followers
- list of followed users
- list of post-likers

Anonymous requests sent for
- the user feed
- tag feed
- location feed
- comments feed
- the number of followers
- and the number of followings
are all limited to various extent. Once a limit is exceeded, further anonymous queries of the given type sent from the same IP address will be rejected by Instagram for an unknown period of time (ranging from hours to days). These rate limits have been set extremely low since around the ides of January 2021.

Occasionally -- and for certain areas / IP addresses permanently -- the very basic anonymous search feature would end up disabled as well, rendering this tool almost entirely useless as long as the block persists for the IP. Except maybe if you happen to know the exact user-ID to the profile you wish to visit, in which case you should type the (presumed) username, followed by a forward slash and the numeric ID, at the search bar, in order to attempt a direct jump to the profile page.

If you want to navigate on this site through the browser's address bar, keep in mind that for Chromium-based browsers you will have to explicitly type out the protocol (https://) in front of the address every single time, otherwise these browsers would perform a web search rather than refreshing the page with the new content. The hash (#) on the URL-line is what's causing this behavior -- tha hash sign is used to differentiate between the various type of contents as the very same html page is getting re-read and subsequently re-composed via javascript.

If anything is out of order, there are links to 3 different websites to try to view the desired page through them.
